# React VR Instagram

`WebVR` experiment on building Instagram UI with `WebGL`

## Installation

Clone the repo

``` bash
git clone https://github.com/petetnt/react-vr-instagram
```

Install deps

``` bash
yarn

# or 

npm install
```

Run the app

```
yarn start

# or

npm start
```

The app should be running at [http://localhost:9666](http://localhost:9666)
